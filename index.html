PROJECTS

Predictive Text App

UC Merced Internship | May 2026

This project was developed to analyze user typing behavior and explore how predictive text features can enhance the text entry experience. Rather than focusing solely on generating word suggestions, the application captures detailed interaction data to better understand how users communicate and interact with a text editor. The goal of this project was to design and implement an interactive text analysis application that records user behaviors throughout a typing session. The system tracks spelling and grammar mistakes, predictive text suggestions, and keyboard interactions such as shortcut usage and editing actions. Once the user finishes typing, the application compiles these interactions into a metrics dashboard that summarizes typing patterns, editing behavior, and overall text entry performance. By combining real-time text analysis with behavioral metrics, this project demonstrates how user interaction data can provide valuable insights into typing habits and interface usability. It integrates human-computer interaction principles, event-driven software development, and data visualization to create an application that supports both user behavior analysis and intelligent text assistance..

Dyadic Association

UC Merced Research | May 2025

Human conversations are driven by associative thinking, where one idea naturally leads to another through semantic and cognitive relationships. Understanding these patterns is essential for developing conversational AI systems that communicate in ways that feel coherent, intuitive, and aligned with human reasoning. This research project explored how large language models emulate human associative behavior through dyadic word association. I contributed to the development of an automated system that orchestrated multi-turn interactions between two AI agents, ensuring stable turn-taking and consistent conversational flow under controlled experimental conditions. The system leveraged LLM APIs to simulate human association patterns while maintaining reproducible dialogue sequences for behavioral analysis. Alongside implementation, I transcribed and structured human dyadic association sessions to produce clean datasets for computational modeling and comparison. By combining cognitive science, natural language processing, and software engineering, this research contributes to a deeper understanding of the similarities and differences between human and AI language behavior. The project demonstrates how carefully designed conversational systems can support the study of semantic reasoning and inform the development of more natural and human-centered AI interactions.

BART Track Inspection

Bay Area Rapid Transit (BART) Internship | May 2025

This project was developed as part of my capstone work to improve the efficiency and reliability of track inspections for Bay Area Rapid Transit (BART). Track inspections are traditionally performed manually, which can be time-consuming and prone to human error. The goal of this project was to design and implement an automated system capable of analyzing track inspection videos and identifying missing or damaged cover boards. The system processes uploaded inspection videos, extracts frames, applies a machine learning model to detect anomalies, and generates structured reports summarizing findings. By automating this workflow, the project aims to reduce manual effort, improve consistency, and provide actionable insights for maintenance teams. This project integrates computer vision, cloud-based machine learning services, and user interface development into a complete end-to-end pipeline
